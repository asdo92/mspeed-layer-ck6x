### Configuration to use ISO German Layout (QWERTZ).

List of key combinations:

    - CapsLock + w = Up
    - CapsLock + s = Down
    - CapsLock + a = Left
    - CapsLock + d = Right
    - CapsLock + k = Home
    - CapsLock + l = End
    - CapsLock + m = PageUp
    - CapsLock + n = PageDown
    - CapsLock + Esc = ยบ
    - CapsLock + Esc + Shift = ^
    - CapsLock + 1 = F1
    - CapsLock + 2 = F2
    - CapsLock + 3 = F3
    - CapsLock + 4 = F4
    - CapsLock + 5 = F5
    - CapsLock + 6 = F6
    - CapsLock + 7 = F7
    - CapsLock + 8 = F8
    - CapsLock + 9 = F9
    - CapsLock + 10 = F10
    - CapsLock + i = F11
    - CapsLock + o = F12
    - CapsLock + Backspace = Delete
    - CapsLock + h = Volume Up
    - CapsLock + g = Volume Down
    - CapsLock + f = Volume Mute
    - CapsLock + p = PrintScreen

List of extra key combinations:

    - Alt + y = <
    - Alt + x = >
    - AltGr + Shift (Left) + y = <
    - AltGr + Shift (Left) + x = >
    - AltGr + y = <
    - AltGr + x = >
