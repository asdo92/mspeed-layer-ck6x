### Configuration to use ISO German Layout (QWERTZ).

List of key combinations:

    - Super + w = Up
    - Super + s = Down
    - Super + a = Left
    - Super + d = Right
    - Super + k = Home
    - Super + l = End
    - Super + m = PageUp
    - Super + n = PageDown
    - Super + Esc = ยบ
    - Super + Esc + Shift = ^
    - Super + 1 = F1
    - Super + 2 = F2
    - Super + 3 = F3
    - Super + 4 = F4
    - Super + 5 = F5
    - Super + 6 = F6
    - Super + 7 = F7
    - Super + 8 = F8
    - Super + 9 = F9
    - Super + 10 = F10
    - Super + i = F11
    - Super + o = F12
    - Super + Backspace = Delete
    - Super + h = Volume Up
    - Super + g = Volume Down
    - Super + f = Volume Mute
    - Super + p = PrintScreen

List of extra key combinations:

    - Alt + y = <
    - Alt + x = >
    - Alt + Esc = \
